В точках максимума или минимума функции угловой коэффициент (производная функции равняется нулю:

k = y'(x) = 0

Чтобы определить максимум в данной точке или минимум, необходимо вычислить значение второй производной для данной точки.

Если y''(x) > 0, в данной точке - минимум функции, если y''(x) < 0 - максимум.
