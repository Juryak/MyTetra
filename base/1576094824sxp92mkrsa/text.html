1. Shape: *.shp

- формат данных ESRI;

- файл - один класс объектов: точечный, линейный или полигональный;

- геометрия и атрибуты могут редактироваться в ArcGIS;

- может быть конвертирован в gdb.


2. Покрытие:

- формат данных ESRI;

- классы объектов: точек, линий, полигонов или аннотаций (текст);

- несколько классов объектов задают отдельный набор объектов;

- геометрия не редактируется в ArcGIS (атрибуты могут быть обновлены);

- может быть конвертирован в gdb.


3. Файлы САПР (CAD):

- не являются форматом данных ESRI;

- классы объектов: точки, линии, полигоны, мультипатчи или аннотаций

- несколько классов объектов задают отдельный чертёж САПР;

- геометрия не редактируется в ArcGIS;

- может быть конвертирован в gdb.


4. Другие растровые форматы:

формат данных ESRI:

- ESRI GRID;

не являются форматом данных ESRI:

- DEM;

- ERDAS IMAGINE;

- MrSID;

- TIFF;

- JPEG;

- GIF;

- множество других.
