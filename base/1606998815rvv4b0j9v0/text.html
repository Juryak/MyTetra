Для обеспечения контраста шрифтов используются:


- размер шрифтов (кегль);


- насыщенность (толщина штрихов) - её контраст является одним из наиболее эффективных способов организации информации;


- структура: не используйте два шрифта из одной категории на одной странице;


- форма: противопоставление строчных букв прописным или прямого начертания курсивному является контрастом форм;


- направление: для расположения текста под наклоном следует иметь чёткую причину и не располагать такой текст по углам. Контраст направлений можно создать сочетанием горизонтальный-вертикальный, горизонтальный-узкие столбцы, подчёркиванием графикой или линиями;


- цвет: создать цветовой контраст можно даже у чёрно-белого изображения - уменьшенные интервалы и жирное начертание визуально темнее обычного, а увеличенные интервалы добавляют яркости.


Для получения максимального эффекта, следует использовать более одного способа обеспечения контраста.
