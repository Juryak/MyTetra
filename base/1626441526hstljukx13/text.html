В атрибутивной таблице shp названия полей должны совпадать с именами атрибутов блока.


Последовательность действий:


1. Открыть атрибуивную таблицу слоя и добавить поле CADType. Сделать его текстовым.

2. В калькуляторе поля в поле Expression ввести "INSERT" (включая кавычки) - значение должно добавиться во все записи.

3. Добавить текстовое поле RefName.

4. В каклькуляторе поля в поле Expression ввести "Название блока"

5. Открыть инструмент Export to CAD.

6. Перетащить на поле Input Features требуемый слой.

7. В поле Output Type выбрать требуемый формат файла.

8. В поле Output File задать имя и расположение файла

9. Перетащить в поле Seed File файл dwg, содержащий определяемый блок.
