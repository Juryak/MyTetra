Рекомендуемые параметры видео


4к - 12-15 fps - P - W - 16/17:9

Используется со штативом, для фото из видео


2.7к - 24-30 fps - S/P - W - 16/17:9

Используется со штативом


1440p - 24-48 fps - S/P - W - 4:3

Используется для видео в движении


1080p - 20-60 fps - S/P - W/M/L - 16:9

Используется в любых условиях


960p - 48-100 fps - S/P(100) - W - 4:3

Используется для замедленных съёмок


720p - 60-120 fps - S/P(120) - W/M/L - 16:9

Используется для замедленных съёмок с рук


WVGA - 240 fps - S - W - 16:9

Используется для сверхзамедленной съёмки, если разрешения достаточно
